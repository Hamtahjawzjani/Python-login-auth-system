# Python Login Authentication System

A simple Python-based login authentication system built for beginners.  
This project demonstrates basic concepts of Python programming such as functions, loops, conditionals, and user input handling.

---

## 📌 Features

- Username and password authentication
- Limited login attempts (3 attempts)
- Automatic program lock after failed attempts
- Clear success and error messages
- Simple and clean console-based interface

---

## 🧠 How It Works

1. The user is prompted to enter a username and password.
2. The system checks if the credentials match the predefined values.
3. If correct:
   - Access is granted.
4. If incorrect:
   - The user loses one attempt.
   - Remaining attempts are displayed.
5. After 3 failed attempts:
   - The program locks automatically.
